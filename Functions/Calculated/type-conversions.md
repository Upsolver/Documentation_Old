# HEX_TO_DECIMAL

Converts a hexidecimal string value to it's corresponding decimal number

{% raw %}

|input|result|
|-----|------|
|`"###"`|`null`|
|`"ff"`|`255`|
|`"0xff"`|`255`|

{% endraw %}
